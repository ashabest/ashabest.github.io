About

This site was made as my final project for UC Berkeley’s Web Design DeCal, which was to make our first fully functional website!

As you can see, this one’s basically about oranges. But why?

I was working with the R language for a different class I was taking at that time, and it had a neat builtin dataset about orange tree growth. In an earlier project we did for the DeCal, my group and I decided to use a deep red-violet to orange gradient to create a "modern" looking background for a lyrics site redesign.

I figured the natural next step was to make a site about oranges. Now I could use that same color scheme and feature some stats about colorful fruits.

The source code for this site features jQuery for the main scripting, and Chart.js for the bar graphs.

Enjoy! ❀
